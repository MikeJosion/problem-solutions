# 2.1 循环链表的连接

## 题目描述

有两个循环单链表，链表头指针分别为 h1 和 h2，编写一个函数将链表 h2 链接到链表 h1 之后，要求链接后的链表仍保持循环链表形式。

## 思路分析

1. 分别找到两个循环链表的**尾节点**（即 `next` 指向头节点的那个节点）
2. 将 h1 的尾节点指向 h2 的第一个数据节点（`h2->next`）
3. 将 h2 的尾节点指向 h1 的头节点，形成新的循环
4. 释放 h2 的头节点（本例中未释放，可优化）

### 图示

```
连接前:
  h1 -> [1] -> [2] -> [3] -> h1（循环）
  h2 -> [4] -> [5] -> [6] -> h2（循环）

连接后:
  h1 -> [1] -> [2] -> [3] -> [4] -> [5] -> [6] -> h1（循环）
```

## 关键代码

```c
LinkList CircularList(LinkList h1, LinkList h2)
{
    LNode *p1 = h1->next, *p2 = h2->next;
    // 找到 h1 的尾节点
    while (p1->next != h1)
        p1 = p1->next;
    // 找到 h2 的尾节点
    while (p2->next != h2)
        p2 = p2->next;
    // h1 尾 -> h2 首个数据节点
    p1->next = h2->next;
    // h2 尾 -> h1 头节点，重新形成循环
    p2->next = h1;
    return h1;
}
```

## 复杂度

- **时间复杂度**: O(n + m)，n 和 m 分别为两个链表的长度
- **空间复杂度**: O(1)
